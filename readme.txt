== Installation ==

= Minimum Requirements =

* PHP 7.2 or greater is recommended
* MySQL 5.6 or greater is recommended
* Ghostscript(GS) must be install and enable on the server https://www.ghostscript.com/doc/current/Install.htm


= Automatic installation =

Automatic installation is the easiest option -- WordPress will handles the file transfer, and you won’t need to leave your web browser. To do an automatic install of SMSA Shipping, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
 
In the search field type “SMSA Shipping,” then click “Search Plugins.” Once you’ve found us,  you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

= Manual installation =

Manual installation method requires downloading the SMSA Shipping plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains [instructions on how to do this here](https://wordpress.org/support/article/managing-plugins/#manual-plugin-installation).


= Woocommerce Addon =

SMSA Shipping (Official) is a Woocommerce addon plugin that extends the shipping functionality of the Woocommerce store. After installation, you have to fill in the SMSA credentials under the woocommerce shipping settings page.

= Get SMSA Account detials =

SMSA sipping (Official) plugin requires an API key. Sign in to your SMSA E-Commerce dashboard and navigate to API documents to copy the API key. If you don't have access to SMSA E-Commerce Dashboard, please email us at info@smsaexpress.com to enable you to ship via SMSA.
